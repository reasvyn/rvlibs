# Rust in Indonesia

The growing Rust community in Indonesia — from early adopters to local events and open source contributions.

## Prerequisites

- [History](history.md) — Rust's origins and global growth

## Early Days

The Indonesian Rust community began forming around 2017–2018, primarily through online forums and social media. Early adopters were individuals exploring Rust for systems programming, embedded development, and web assembly. The community grew organically through:

- **Telegram Group** — "Rust Indonesia" became the primary discussion hub
- **GitHub** — Indonesian developers contributing to Rust open source projects
- **Personal blogs** — A growing number of Rust tutorials and articles written in Bahasa Indonesia

## Community Growth

The community has grown significantly:

- **Rust Indonesia Telegram** — Hundreds of active members discussing Rust daily
- **Local meetups** — Regular in-person gatherings in Jakarta, Bandung, Yogyakarta, and Surabaya
- **Tech communities** — Rust talks at Indonesia's major tech conferences (PyCon ID, TechInAsia, DevCon)
- **University outreach** — Rust workshops at universities (UI, ITB, UGM, ITS)
- **Open source** — Indonesian developers contributing to Rust ecosystem projects

## Notable Contributions

Indonesian developers have contributed to:

- **Rust translations** — Community-driven Bahasa Indonesia translations of The Rust Book
- **Ecosystem crates** — Libraries and tools published by Indonesian Rust developers
- **Bug reports and patches** — Contributions to rustc, Cargo, and popular crates
- **Community content** — Blog posts, tutorials, and YouTube content about Rust in Bahasa Indonesia

## Key Figures

Several Indonesian developers have become recognised in the global Rust community:

- **Local maintainers** — Indonesians maintaining popular Rust crates
- **Speakers** — Indonesian developers presenting at international Rust conferences
- **Educators** — Rust trainers and workshop facilitators in Indonesia

## How to Get Involved

- **Join the Telegram group** — Search "Rust Indonesia" on Telegram
- **Attend meetups** — Follow local tech community calendars for Rust events
- **Contribute** — Start with good-first-issue labels on crates you use
- **Write** — Share your Rust experience in Bahasa Indonesia to grow the local knowledge base
- **Present** — Propose a Rust talk at local tech conferences
