# Rust in China

One of the largest and fastest-growing Rust communities in the world.

## Prerequisites

- [History](history.md) — Rust's origins and global growth

## Community Scale

China has one of the largest Rust communities globally:

- **Wx Rust (WeChat)** — Major Rust discussion groups with thousands of members
- **Rust.cc** — Chinese-language Rust forum
- **Zhihu** — Extensive Rust content on China's Quora-like platform
- **Bilibili** — Hundreds of Rust tutorial videos on China's YouTube-equivalent

## Corporate Adoption

Chinese tech giants have embraced Rust:

- **Alibaba** — Uses Rust in cloud infrastructure, database systems (PolarDB), and service mesh (MOSN written in Rust).
- **Tencent** — Adopts Rust for game engine tooling, cloud services, and performance-critical backend components.
- **ByteDance (TikTok/Douyin)** — Rust used in performance-critical infrastructure and media processing.
- **Huawei** — Researching Rust for HarmonyOS, networking equipment, and cloud services.
- **PingCAP** — The company behind TiDB uses Rust for core storage engine components.
- **Ant Group** — Uses Rust in blockchain and financial infrastructure.

## Open Source Contributions

Chinese developers are significant contributors to:

- **Rust compiler and tools** — Regular contributions to rustc, Cargo, and the Rust project
- **Tokio ecosystem** — Active maintenance and contributions
- **Database infrastructure** — TiKV (distributed KV store) is written in Rust by PingCAP
- **Cloud-native tooling** — Contributions to Kubernetes-related Rust projects
- **Rust documentation** — Chinese translations of The Rust Book, Rust by Example, and other resources
