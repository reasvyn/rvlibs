# Rust in Brazil

One of Latin America's most active Rust communities — known for community-driven initiatives and growth.

## Prerequisites

- [History](history.md) — Rust's origins

## Community

- **Rust Brasil** — The primary Brazilian Rust community on Telegram (thousands of members)
- **Rust Prog BR** — A discussion group for Portuguese-language Rust content
- **Reddit** — r/rust_brasil for community discussions and questions
- **Medium / Dev.to** — Brazilian developers publishing Rust content in Portuguese

## Corporate Adoption

- **Nubank** — Brazil's largest fintech uses Rust in its core banking infrastructure and fraud detection systems
- **Stone** — Payment processing company exploring Rust for performance-critical systems
- **iFood** — Food delivery giant evaluating Rust for backend services
- **VTEX** — E-commerce platform using Rust in infrastructure components
- **CloudWalk** — Fintech startup using Rust in payment processing

## Community Initiatives

- **Rust Brasil Conference** — Annual online/in-person conference featuring Brazilian and international speakers
- **Rustaceans BR** — Twitch streams and YouTube content about Rust in Portuguese
- **Translation projects** — Portuguese translations of The Rust Book and other essential resources
- **Meetups** — Regular gatherings in São Paulo, Rio de Janeiro, Belo Horizonte, Porto Alegre
- **Mentorship** — Community-led mentoring programmes for new Rust developers

## Notable Contributions

- Brazilian developers active in the global Rust community
- Rust in Portuguese educational content — one of the largest collections of Portuguese-language Rust material
- Community tools — Automation and tools built by the Brazilian Rust community to support new learners
