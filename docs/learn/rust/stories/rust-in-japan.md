# Rust in Japan

Japan's Rust community — from embedded systems to web development, and one of the most active Rust communities in Asia.

## Prerequisites

- [History](history.md) — Rust's origins

## Community

- **Rust Tokyo** — The largest regular Rust meetup in Japan, featuring talks from local and international speakers
- **Rust Kansai** — Community group in the Osaka/Kyoto/Kobe region
- **Rust JP** — The Japanese Rust community's primary online hub
- **Zenn** — Japanese developers publishing extensive Rust articles on the Zenn platform

## Corporate Adoption

- **Fujitsu** — Researching Rust for embedded systems and high-performance computing
- **Sony** — Rust used in embedded and multimedia processing
- **Recruit** — Using Rust for backend systems and developer tooling
- **DeNA** — Rust in game infrastructure and web services
- **Mercari** — Japan's largest flea market app uses Rust in backend services
- **Preferred Networks** — Deep learning infrastructure and compilers in Rust

## Unique Contributions

- **Embedded Rust** — Japan has a strong embedded systems industry, and Japanese developers contribute significantly to embedded Rust tooling (probe-rs, embedded-hal)
- **Game development** — Japanese game studios exploring Rust for tooling and runtime systems
- **Research** — Rust being explored for high-performance computing at Japanese universities (University of Tokyo, Kyoto University)
