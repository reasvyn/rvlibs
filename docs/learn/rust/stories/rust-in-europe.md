# Rust in Europe

A diverse and mature Rust ecosystem across European countries — from research to industry adoption.

## Prerequisites

- [History](history.md) — Rust's origins

## Germany

Germany has one of the most active Rust communities in Europe:

- **Rust Cologne** — One of the longest-running Rust meetups in Europe
- **Rust Berlin** — Very active meetup group at the European Rust hub
- **Rust Munich** — Regular meetups with talks and workshops
- **Rust Hamburg** — North German Rust community
- **Ferrocene** — A safety-qualified Rust compiler developed by **Ferrous Systems** (Berlin)
- **Corporate adoption** — Siemens, Bosch, Volkswagen exploring Rust for embedded and automotive

## United Kingdom

- **Rust London** — Largest UK Rust meetup with frequent talks and socials (2000+ members)
- **Rust Cambridge** — Academic and industry Rust community at a tech hub
- **Rust Edinburgh** — Scottish Rust community
- **Corporate adoption** — Google (UK offices), ARM, Improbable using Rust in production

## France

- **Rust Paris** — Active meetup group
- **Rust Bordeaux** — Growing Rust community in southwestern France
- **Corporate adoption** — Dassault Systèmes, Airbus exploring Rust for safety-critical systems
- **Research** — INRIA (French National Research Institute) actively researching Rust for formal verification

## Netherlands

- **Rust Amsterdam** — Active meetup group in the Dutch tech capital
- **RustNL** — Annual Rust conference in the Netherlands
- **Corporate adoption** — TomTom, Adyen using Rust in production
- **Research** — Delft University of Technology researching Rust in cyber-physical systems

## Nordic Countries

| Country | Highlights |

## Other Notable Regions

- **Switzerland** — Rust Zurich meetup, ETH Zurich researching Rust
- **Italy** — Rust Rome/Milan meetups, RustLab conference in Florence
- **Spain** — Rust Barcelona/Madrid meetups, growing community in the Spanish tech scene
- **Poland** — Rust Warsaw meetup, growing corporate adoption
- **Portugal** — Rust Lisbon meetup, increasing academic interest
