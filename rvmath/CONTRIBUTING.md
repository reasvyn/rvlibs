# Contributing to rvmath

Thank you for your interest in rvmath! We welcome contributions to make this a comprehensive mathematics library for Rust.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/reasvyn/rvmath.git
    ```
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make your changes**, ensuring they follow the project's coding standards.
5.  **Write and run tests**:
    ```bash
    cargo test
    ```
6.  **Commit your changes** with clear and concise messages.
7.  **Push to your fork** and **submit a pull request**.

## Development Guidelines

- **Type Safety**: Prioritize Rust's type system to ensure dimensional and mathematical correctness.
- **Testing**: Every new feature or bug fix must include corresponding tests using `rvtest` in the `tests/` directory.
- **Documentation**: Provide clear comments and keep the README up to date.

## Roadmap

The current focus is on the **Unit System (v0.1.0)**. Future updates will include:
- Linear Algebra (Vectors, Matrices)
- Calculus and Differential Equations
- Statistics and Probability
- Physics constants and formulas

Join us in building a robust math engine for the Rust community!
